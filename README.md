# Cybersecurity Keylogger Lab

## Overview

Cybersecurity Keylogger Lab is an educational Python project that demonstrates how keyboard events can be captured and logged using the `pynput` library.

This project was created for learning purposes to understand keyboard event handling, file operations, and cybersecurity concepts related to monitoring and logging user input.

## Features

* Real-time keyboard event capture
* Logs keystrokes to a text file
* Handles special keys (Enter, Space, Backspace)
* Simple and lightweight implementation
* Easy to understand for beginners

## Technologies Used

* Python
* pynput

## Installation

```bash
pip install pynput
```

## Usage

```bash
python main.py
```

The program will create and update a log file containing captured keyboard events.
